# Тестовые ресурсы для интеграционных сценариев

Используются в [GraphAgentToolScenariosIntegrationTest](../../kotlin/agent/GraphAgentToolScenariosIntegrationTest.kt).
При запуске тестов копируются во временную папку под `$HOME` (требование isPathSafe тулов).

| Файл | Назначение |
|------|------------|
| `test.txt` | Сценарий 16 (извлечь текст): проверяем наличие «Test content», «проверки», «извлечения». |
| `read_me.txt` | Сценарии 15 (перенос), 18 (открыть файл): ожидаемый текст в ответе. |
| `sample.csv` | Сценарий 11 (график по CSV): колонки name, age, value для построения графика. |
| `directory/file.txt` | Сценарий 13 (список файлов): файл в подпапке. |
| `sample.pdf` | Создаётся в тесте (PDFBox) во временной папке — сценарий 17 (читать PDF постранично). |
